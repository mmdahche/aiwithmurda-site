# START HERE — Founder Finance Pack

You bought six skills that cover the money story: operating metrics, models,
fundraising assets, outreach, and board communication.

## First win in 3 steps (~20 minutes)

1. **Install all six:** `bash install/setup.sh`
2. **Metrics drill:** paste MRR, churn, new logos, S&M spend. Ask the agent to
   run the `saas-metrics-coach` workflow — verdict, top risk, top lever.
3. **Consistency drill:** open `investor-materials` and build the source-of-truth
   table before editing any deck slide.

## Where everything is

- `payload/*.md` — six skills (source copies)
- `install/claude-code/` + `install/codex/` — dual layouts
- `examples/seed-round-materials-walkthrough.md` — memo + deck + outreach chain
- `VERIFY.md` — checklist we ran before shipping

## The one rule

**One source of truth for numbers.** If ARR differs between the deck and the
memo, investors assume both are wrong — fix the ledger before polish.

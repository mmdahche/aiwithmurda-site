# Founder Finance Pack

**CFO-grade analysis and fundraising materials — as six agent skills.**

Six clean-room operator disciplines for founders raising or reporting: startup
finance, statement analysis, SaaS metrics, investor documents, outreach, and
board decks. Markdown-only — no scripts, no cloud APIs.

| Skill | What it does |
|-------|--------------|
| **cfo-advisor** | Burn, runway, unit economics, raise sizing, scenarios |
| **financial-analyst** | Ratios, variance, rolling forecast, DCF framing |
| **saas-metrics-coach** | ARR, churn, NRR, LTV, CAC, benchmarks |
| **investor-materials** | Deck, memo, one-pager with a consistency ledger |
| **investor-outreach** | Cold, warm intro, follow-up, update emails |
| **board-deck-builder** | Board/QBR structure, KPIs, bad-news protocol |

## Time to first value

~20 minutes: install all six, paste last month's numbers into
`saas-metrics-coach`, then run `investor-materials` to build a consistency
ledger before touching the deck.

## Install

**Quick path:** `bash install/setup.sh`

**Manual:** copy each file from `install/claude-code/` or `install/codex/<name>/`.

## Provenance

Clean-room rewrite (2026-07-16). Provenance audit found two ECC-tagged reference

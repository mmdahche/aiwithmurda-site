# Investor Materials

You build fundraising assets that **cross-reference cleanly** — the ARR in the
deck matches the memo matches the use-of-funds table. Credibility is consistency.

## When to use

- Drafting or revising a pitch deck (seed through Series A)
- Writing a one-pager or investor memo
- Building use-of-funds and 18-month milestone plan
- Updating materials after a metrics change (post-close refresh)
- Accelerator or grant application with financial attachments

## Document set

| Asset | Length | Job |
|-------|--------|-----|
| One-pager | 1 page | Hook + traction + ask + contact |
| Deck | 10–15 slides | Narrative arc for live meeting |
| Memo | 2–4 pages | Async deep read for partners |
| Data room index | List | What exists, where, freshness date |

Pick the minimum set for the round — don't ship five stale variants.

## Deck arc (default)

1. Problem — specific, costly, urgent for a defined buyer
2. Solution — what you ship today, not roadmap fantasy
3. Why now — market shift, regulation, cost curve, behavior change
4. Traction — metrics with dates; chart > table when possible
5. Business model — how money flows, gross margin direction
6. Market — bottom-up sizing; show the math
7. Competition — honest 2×2; your wedge in one sentence
8. Team — why *this* team for *this* problem
9. Financials — 12–18 mo forecast summary + key assumptions
10. Ask — amount, instrument, use-of-funds, milestones unlocked
11. Appendix — detailed metrics, product screenshots, hiring plan

## Consistency ledger

Before finalizing, build a single **source-of-truth table**:

|-------|-------|------------------|---------------|

Every asset copies from this table — no hand-edited duplicates.

## Use of funds

- Buckets: product, GTM, G&A, buffer (5–10%)
- Tie each bucket to a **milestone** investors can verify in 12 months
- Show headcount plan if >50% goes to people — role, start quarter, loaded cost

## Writing rules

- Claims need evidence: "leading" → cite ranking or remove
- Projections labeled **forward-looking assumptions**, not facts
- Round terms only when approved for external share
- Same company name, logo, and metric definitions everywhere

## Output rules

- Deliver outline + speaker notes when deck requested
- Flag internal contradictions before polish
- If numbers missing, stub `[TBD — need X from finance]` not invented figures

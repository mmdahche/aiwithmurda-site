## What's inside

| File | What it does |
|---|---|
| `payload/verify-before-claiming.md` | The claim gate: no completion claims without fresh verification evidence run in the same message; never relay a sub-agent's "success" without reading its artifact |
| `payload/inspect-before-edit.md` | The single prompt that makes an agent map, narrate current behavior, propose the smallest change, and name its proof — before touching a file |
| `payload/daily-operator-checklist.md` | The daily rhythm: 15/30/60-minute session tracks, verification gates, the bad-output protocol, a 10-minute weekly review |
| `install/claude-code/` + `install/codex/` | The skill in both agent layouts |

## Setup

Copy `install/claude-code/verify-before-claiming.md` into your skills or
commands folder (Codex: the `install/codex/` layout into `.agents/skills/`).
The script and checklist are reference files — keep them beside your terminal.
No dependencies, nothing to run.

# Content Engine Pack

**Hooks, scripts, and humanized copy — without the fake-guru stink.**

Five clean-room skills plus a faceless-channel workflow. Ethics baked in: no
fabricated income hooks, no fake personas, citations required on trend posts.

| Skill | What it does |
|-------|--------------|
| **hooks-angles** | Short-form hook frameworks |
| **content-humanizer** | Voice rebuild for AI-sounding drafts |
| **ugc-scriptwriter** | Timestamped vertical video scripts |
| **trending-content** | Verified trend → angle with sources |
| **audio-overview** | Spoken chapter scripts from docs |

## Provenance

Clean-room rewrite (2026-07-16). Third-party humanizer and marketing library
skills were **not ported** — failed provenance components rewritten from scratch.

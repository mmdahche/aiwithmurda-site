# UGC Scriptwriter

Author **shot-by-shot scripts** for vertical short-form ads and organic UGC.
Authenticity over polish — handheld energy, not broadcast.

## When to use

- TikTok / Reels / Shorts ad scripts
- Product demo in selfie format
- Before sending prompts to a video generation tool

## 4-beat arc

| Beat | Seconds | Job |
|------|---------|-----|
| Hook | 0–2 | Pattern interrupt — specific, not hype |
| Problem | 2–8 | Relatable pain in one scenario |
| Action | 8–20 | Show product/steps — hands visible |
| Result + CTA | 20–30 | Honest outcome + one CTA |

Adjust timing for 15s vs 30s targets; never pad.

## Voice rules

- Neutral conversational American (adjust per brief)
- First person, present tense where natural
- One idea per sentence spoken aloud
- No scripted laughter or fake surprise

## Shot list format

```markdown
## [Product] — 30s UGC
**Hook (0:00–0:02)** — VISUAL: ... — AUDIO: "..."
**Problem (0:02–0:08)** — ...
**Action (0:08–0:22)** — ...
**CTA (0:22–0:30)** — ...
```

Each shot: framing (selfie / tabletop / screen), lighting note (window side / soft indoor), prop list.

## Visual prompt grammar (for AI video tools)

When output includes generation prompts:
- Subject + action + environment + camera (handheld slight shake)
- Reference `@product` or `@face` placeholders for asset binding
- No celebrity likenesses or misleading before/after unless real

## Ethics

- Real product only — no undisclosed sponsorship voice
- Results labeled typical vs best-case
- No income claims in CTA

## Output deliverables

1. Full script with timestamps
2. B-roll shot list (optional cuts)
3. On-screen text captions (≤6 words per card)

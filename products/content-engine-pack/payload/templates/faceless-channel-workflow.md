# Faceless Channel Workflow

Operator workflow for **faceless or voice-only channels** — no fake guru persona.

## Pipeline

1. **Research** — `trending-content` with 3+ sources cited
2. **Angle** — pick one audience slice; run `hooks-angles` for 3 options
3. **Script** — `ugc-scriptwriter` OR long-form `audio-overview` depending on format
4. **Humanize** — `content-humanizer` pass on spoken lines and captions
5. **Produce** — record VO / generate visuals (your tools)
6. **Publish** — caption file + description with source links where claims appear
7. **Receipt** — save metrics screenshot; note what hook variant shipped

## Quality gates

- [ ] Every stat has a source in description or pinned comment
- [ ] Hook ≤12 words or marked exception
- [ ] No income claims without proof
- [ ] Thumbnail text matches video promise

## Formats

| Format | Primary skills |
|--------|----------------|
| 30s reel | hooks-angles → ugc-scriptwriter → content-humanizer |
| Explainer VO | trending-content → audio-overview |
| Thread | trending-content → hooks-angles → content-humanizer |

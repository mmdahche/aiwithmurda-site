## Seven-Day Implementation Plan

Day 1: Rewrite the session brief for your own workflow.

Day 2: Run the workflow scoper prompt and delete anything that is not part of the smallest useful slice.

Day 3: Build or document the first version.

Day 4: Test the user path and capture before/after proof.

Day 5: Write the proof receipt.

Day 6: Turn the receipt into one post, clip, email, or checkout-page section.

Day 7: Decide whether to iterate, package, or sell the workflow.

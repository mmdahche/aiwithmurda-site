## System audit

Review:

- Project guidance still matches the project.
- Commands are current and verified.
- Protected paths and secret handling remain correct.
- Installed skills are relevant and readable.
- Handoffs contain an exact next action.
- Evidence logs match claimed completion.
- Agent roles are clear.
- Update version and compatibility review are current.
